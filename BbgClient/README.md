# BBGCLIENT

## Description

BBGCLIENT is a TCP client for the Parking-System project.

The client runs on BeagleBone Green and communicates with the Parking-System TCP server.

## Responsibilities

- Establish a TCP connection to the server.
- Send parking requests.
- Receive server responses.
- Handle connection errors and reconnections.

## Current status

Current implementation is used as a temporary Linux test client.

The final version will run on BeagleBone Green and communicate with RFID and other hardware modules.
